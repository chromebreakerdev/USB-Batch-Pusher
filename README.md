# USB Batch Pusher (Windows)

A tiny Tkinter GUI that pushes a selected **file or folder** to **multiple USB drives** at once.
- Detects removable drives (Windows)
- Copies file or whole folder
- Checks free space before copy
- Optional overwrite
- Progress bar + log
